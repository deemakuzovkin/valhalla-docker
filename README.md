## Container launch
### [Search](https://github.com/deemakuzovkin?tab=packages&repo_name=valhalla-docker) image in tags

``` bash 
  docker run -d -it --name valhalla ghcr.io/deemakuzovkin/valhalla-docker/$Tag
```
